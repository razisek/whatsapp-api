# WhatsApp Api

create using package [wa-automate-nodejs](https://github.com/open-wa/wa-automate-nodejs)

## Installation

```bash
$ git clone https://github.com/razisek/whatsapp-api
$ cd whatsapp-api
$ npm start
```
